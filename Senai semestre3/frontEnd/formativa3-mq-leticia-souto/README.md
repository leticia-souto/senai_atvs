# 💻 GalaxyTours - Viagens Espaciais

Este é um projeto desenvolvido para a avaliação formativa de Front-End, com o intuito de avaliar a cacidade dos alunos de adicionar media Queries no site, para que a tela se ajuste conforme o tamanho do dispositivo.
O projeto contém funcionalidades para deixar um ambiente mais agradável para o usuário, isso inclui a capacidade de ajuste dos elementos da tela para cada tipo de dispostivo. 
É um app que permite uma viagens para diversos planetas! 

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
