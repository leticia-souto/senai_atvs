# 💻 FitTrack Saúde e Vida Fitness

Este é um projeto desenvolvido para a avaliação de Front-End, com o intuito de criar interatividade um app web de saúde e vida fitness.
O projeto contém várias funcionalidades para deixar um ambiente mais agradável para o usuário, calculando inclusive a quanridade de água ideal para ser tomada por dia de acordo com o peso. 
É um app que incentiva uma vida mais saudável de maneira diverdita e personalizada.

## 🛠️ Tecnologias Utilizadas
* HTML5 (Semântica e Acessibilidade)
* CSS3 (Estilização e Feedback Visual)
* JavaScript (Manipulação de DOM e Eventos)

## 🚀 Funcionalidades Implementadas
* **Interatividade Visual:** Elementos que reagem ao passar do mouse (Hover e manipulação de Classes).
* **Saudação Dinâmica:** Mensagem de boas-vindas que muda conforme o horário local do usuário.
* **Cálculo em Tempo Real:** Formulário que processa e exibe resultados assim que o usuário digita.
* **Injeção de Conteúdo:** Adição de novos itens na tela utilizando manipulação de DOM.
